Intelligence
// Information Platform Division

We don't aggregate news. We understand it.
Using SBERT embeddings and HDBSCAN clustering, thousands of articles are grouped into meaningful issue clusters — automatically, in real time.

Core Technology
Semantic Embedding maps each article into vector space, capturing meaning beyond keywords. Density-based clustering surfaces emergent issues without predefined categories — letting the news define itself.

// Hardware Division

We build hardware that solves real problems. From custom UART scheduling protocols on ESP32 to wireless analyzers and low-power display systems — firmware and PCB, end to end.

Core Technology
BPU (Batch Processing Unit) is a custom UART backpressure scheduling protocol for ESP32 — designed for reliable high-throughput serial communication in constrained environments.
