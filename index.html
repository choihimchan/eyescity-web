Intelligence
// Information Platform Division

We don't aggregate news. We understand it.
Using SBERT embeddings and HDBSCAN clustering, thousands of articles are grouped into meaningful issue clusters — automatically, in real time. SBERT 임베딩과 HDBSCAN 클러스터링 기반 AI 자동 뉴스 분석 플랫폼 디비전입니다.

Core Technology
Semantic Embedding maps each article into vector space, capturing meaning beyond keywords. Density-based clustering surfaces emergent issues without predefined categories — letting the news define itself.

Embedded
// Hardware Division

We build hardware that solves real problems. From custom UART scheduling protocols on ESP32 to wireless analyzers and low-power display systems — firmware and PCB, end to end. ESP32 기반 커스텀 프로토콜 및 하드웨어 제작(펌웨어, PCB 설계) 디비전입니다.

Core Technology
BPU (Batch Processing Unit) is a custom UART backpressure scheduling protocol for ESP32 — designed for reliable high-throughput serial communication in constrained environments.
